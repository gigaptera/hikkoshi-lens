# HikkoshiLens デザインシステム仕様書

## コンセプト: Urban Living × Sharp Minimalism

**"洗練された都市生活を、数学的な調和で。"**

温かみのある都市的カラーパレット（Teal & Warm Gray）と、数学的比率（黄金比・フィボナッチ数列）に基づいた精密なレイアウトを融合させたデザインシステムです。

---

## 1. デザイン原則

1.  **シャープネス (Sharpness)** - すべての要素の角丸を 0 にし、直線的で精確なフォルムを徹底します。
2.  **ミニマリズム (Minimalism)** - 装飾的な要素を削ぎ落とし、情報の明瞭性と機能性を追求します。
3.  **数学的調和 (Mathematical Harmony)** - 黄金比とフィボナッチ数列をスペーシングやタイポグラフィの基盤とし、無意識の心地よさを生み出します。
4.  **都市的感性 (Urban Aesthetic)** - 都市生活の洗練さを、落ち着いた配色とタイトなレイアウトで表現します。

---

## 2. 数学的比率の適用

本システムでは、視覚的調和を最大化するために以下の比率を統合します。

### 黄金比 (Golden Ratio) φ ≈ 1.618

- **適用先**: タイポグラフィスケール、レイアウトの分割（メイン vs サイドバー）、コンポーネントの縦横比。
- **効果**: 洗練された高級感と、自然な視線誘導を実現。

### フィボナッチ数列 (Fibonacci Sequence)

- **数値**: 8, 13, 21, 34, 55, 89...
- **適用先**: スペーシング（マージン、パディング、ギャップ）。
- **効果**: リズム感のある自然な広がりと、実装上の整合性を両立。

---

## 3. カラーシステム

### Primary - Teal (メイン・アクション)

洗練された印象と信頼感を与えるティール。

- `primary-50`: `#F0FDFA` (背景)
- `primary-500`: `#14B8A6` (メインボタン、リンク)
- `primary-700`: `#0F766E` (ホバー、強調)

### Neutral - Warm Gray (ベース・構造)

都市のコンクリートや洗練されたインテリアを想起させるウォームグレー。

- `neutral-50`: `#FAFAF9` (全体の背景)
- `neutral-200`: `#E7E5E4` (ボーダー、区切り線)
- `neutral-500`: `#78716C` (本文テキスト)
- `neutral-900`: `#1C1917` (見出し、強調)

### Accent - Orange (アクセント)

エネルギーと注意を引くためのオレンジ。

- `accent-500`: `#F97316` (特別な通知、ハイライト)

---

## 4. スペーシング (Fibonacci)

Tailwind v4 の `--spacing-*` 変数にフィボナッチ数列をマッピングします。

| 変数名       | 値 (rem)    | 値 (px) | 用途                         |
| :----------- | :---------- | :------ | :--------------------------- |
| `spacing-2`  | `0.5rem`    | 8px     | 最小単位、要素内余白         |
| `spacing-3`  | `0.8125rem` | 13px    | ボタン内、小さなカードの余白 |
| `spacing-5`  | `1.3125rem` | 21px    | コンテンツグループ間の余白   |
| `spacing-8`  | `2.125rem`  | 34px    | カード内、セクション内余白   |
| `spacing-13` | `3.4375rem` | 55px    | セクション間の大きな余白     |

---

## 5. タイポグラフィ (Golden Ratio)

16px をベースとし、黄金比（x 1.618）でスケールさせます。

| レベル      | フォントサイズ     | 比率 | 用途                 |
| :---------- | :----------------- | :--- | :------------------- |
| **Display** | `4.236rem` (~68px) | φ³   | ヒーローコピー       |
| **H1**      | `2.618rem` (~42px) | φ²   | ページタイトル       |
| **H2**      | `1.618rem` (~26px) | φ    | セクション見出し     |
| **H3**      | `1.25rem` (20px)   | -    | サブ見出し（調整値） |
| **Base**    | `1rem` (16px)      | 1    | 本文                 |
| **Small**   | `0.875rem` (14px)  | -    | 補助テキスト         |
| **XS**      | `0.618rem` (~10px) | 1/φ  | 注釈、キャプション   |

**フォントファミリー:**

- 英数字: `Inter`
- 日本語: `Noto Sans JP`

---

## 6. 基本コンポーネント定義

### 角丸 (Border Radius)

- **常に 0**: `border-radius: 0;`

### ボタン (Button)

- 特徴: 2px のボーダー、シャープな角、フィボナッチパディング (`px-5 py-3`)。
- アニメーション: 150ms のカラー変化。

### カード (Card)

- 特徴: 黄金比の縦横比 (`aspect-[1.618/1]`)を基本とし、`neutral-200` の繊細なボーダーと、シャープなシャドウ。

## 7. 実装コード (Tailwind v4 CSS @theme)

```css
@theme {
  /* Colors */
  --color-primary-500: #14b8a6;
  --color-neutral-900: #1c1917;

  /* Spacing - Fibonacci */
  --spacing-2: 0.5rem; /* 8px */
  --spacing-3: 0.8125rem; /* 13px */
  --spacing-5: 1.3125rem; /* 21px */
  --spacing-8: 2.125rem; /* 34px */
  --spacing-13: 3.4375rem; /* 55px */

  /* Typography - Golden Ratio */
  --font-size-2xl: 2.618rem; /* ~42px */
  --font-size-xl: 1.618rem; /* ~26px */

  /* Radii */
  --radius-none: 0;
  --radius-DEFAULT: 0;
}
```
