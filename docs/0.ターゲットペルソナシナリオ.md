## 基本情報
- 年齢: 22
- 職業: 学生（内定先あり・来春入社予定）
- 現居: 大阪の実家（生活水準の基準点）
- 状況: 東京での一人暮らし検討中。勤務地は大手町。

## シナリオフロー

1. **サービス認知 (Index)**:
   「今の家の便利さ、東京でも維持できますか？」というLPを見て、実家と比較できる点に惹かれて利用を開始する。

2. **診断と基準作成 (Test)**:
   ライフスタイル診断に答え、最後に実家の住所を入力。「実家のスコア：72点（治安◎、利便性△）」という結果が出る。これが東京探しの基準になる。

3. **勤務地検索 (Search/office)**:
   勤務地「大手町」を入力。システムが通勤30〜40分圏内の駅を洗い出し、大手町への通勤時間を加味したスコアを計算する。

4. **候補の絞り込み (Search/nearby)**:
   地図上に「82」「78」「65」といったスコア付きのピンが並ぶ。「82」と表示された「清澄白河駅」のピンをタップし、**駅詳細ページ**へ飛ぶ。
   駅ごとの家賃相場や治安を見て、良さそうなので「比較リスト」に追加する。

5. **比較検討 (Compare)**:
   比較画面で「大阪の実家」と「清澄白河」を並べる。「家賃は上がるが、通勤は実家より楽、スーパーの数は同じくらい」と納得する。

6. **詳細な地点評価 (Search/home or Map Tap)**:
   清澄白河エリアで、SUUMOで見つけた具体的な物件の住所を入力（または地図上でピン刺し）。
   **地点詳細ページ**が表示され、「駅スコアは高いが、この物件の周りはコンビニまで徒歩10分」という判定が出る。「ここは不便だからやめよう」と判断し、駅に近い別の地点を探す。

## ゴール
数値化された「実家」という基準を持つことで、土地勘のない東京でも自信を持って「ここは住みやすい」「ここは不便」と判断できる状態になること。
